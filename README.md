# Builder Atlas

**Live:** [builderatlas.lovable.app](https://builderatlas.lovable.app)

A community-curated leaderboard of active builders, ranked by their GitHub commit activity. Discover who's shipping the most code this week.

## Features

- **Live commit tracking** — Fetches weekly commit counts from the GitHub Events API via a backend function
- **Leaderboard ranking** — Builders sorted by commits per week, with a "Most Active" spotlight
- **Tag filtering** — Filter builders by tags like AI, Web3, DevTools, etc.
- **Community submissions** — Anyone can submit a builder with their GitHub profile and project URL

## Tech Stack

- React + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Lovable Cloud (database, edge functions)
